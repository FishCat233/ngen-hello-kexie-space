## 简介

APP开发是指开发移动设备上的应用程序，包括 Android 和 iOS 平台。移动应用开发涉及原生开发、跨平台开发等多种技术方案。

## 学习内容

### Android 原生开发
- Java / Kotlin
- Android SDK
- Android Jetpack
- Material Design

### iOS 开发
- Swift / Objective-C
- iOS SDK
- SwiftUI / UIKit
- App Store 发布流程

### 跨平台开发
- Flutter（Dart）
- React Native（JavaScript）
- uni-app / Taro
- 原生模块集成

### 移动端特性
- 推送通知
- 地理位置服务
- 相机与多媒体
- 本地存储

## 发展方向

- Android/iOS 开发工程师
- 跨平台开发工程师
- 移动端架构师
- 移动产品经理
