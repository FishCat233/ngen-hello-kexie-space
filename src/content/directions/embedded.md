## 简介

硬件开发（嵌入式开发）是指开发嵌入式系统，将计算机技术应用到各种设备中。嵌入式系统广泛应用于消费电子、工业控制、物联网等领域。

## 学习内容

### 硬件基础
- 数字电路与模拟电路
- 微处理器架构
- 传感器与执行器
- PCB 设计基础

### 嵌入式编程
- C / C++
- 汇编语言
- 实时操作系统（FreeRTOS、RT-Thread）
- 驱动开发

### 开发平台
- Arduino
- STM32
- ESP32
- Raspberry Pi
- 树莓派 Pico

### 通信协议
- UART / SPI / I2C
- CAN 总线
- Modbus
- MQTT（物联网）

## 发展方向

- 嵌入式软件工程师
- 硬件工程师
- 物联网开发工程师
- 自动化工程师
