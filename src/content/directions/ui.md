## 简介

UI（User Interface）设计是指用户界面设计，关注产品的视觉呈现和交互方式。好的 UI 设计能够提升用户体验，让产品更易用、更美观。

## 学习内容

### 设计基础
- 色彩理论
- 排版与字体
- 构图与布局
- 设计原则

### 设计工具
- Figma
- Sketch
- Adobe XD
- Photoshop / Illustrator

### 界面设计
- 移动端界面设计
- Web 界面设计
- 图标设计
- 设计系统构建

### 交互设计
- 用户体验（UX）基础
- 交互流程设计
- 原型制作
- 用户测试

## 发展方向

- UI 设计师
- UX 设计师
- 产品设计师
- 视觉设计师
