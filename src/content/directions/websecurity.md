## 简介

Web安全是指保护 Web 应用程序免受各种攻击的技术和实践。随着互联网的发展，Web 安全问题日益重要，涉及 SQL 注入、XSS、CSRF 等多种攻击类型。

## 学习内容

### Web 基础
- HTTP/HTTPS 协议
- Web 服务器配置
- 会话管理
- 认证与授权

### 常见漏洞
- SQL 注入
- XSS（跨站脚本攻击）
- CSRF（跨站请求伪造）
- 文件上传漏洞
- 命令注入

### 安全测试
- 渗透测试方法论
- Burp Suite 使用
- SQLMap
- 漏洞扫描工具

### 防御技术
- 安全编码规范
- WAF（Web 应用防火墙）
- 安全加固
- 日志与监控

## 发展方向

- Web 安全工程师
- 渗透测试工程师
- 安全审计师
- 安全开发工程师
